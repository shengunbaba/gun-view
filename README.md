## 使用

- 直接复制需要的组建到你的项目里面

## 为什么写这个

- ant-design 是写管理后台的ui解决方案，且比较重。
- 但平时写网站， 或是专题页等其余产品， 需要一个拿来即用，方便调节和配置的组件。
